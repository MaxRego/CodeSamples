README for DrosophilaMelanogaster.pl

This perl script parses the dmel.gff datafile from flybase and populates a database with useful data

NOTE: This script takes hours to run and complete, to completely populate the database 5+ hours on a normal machine

Required Modules:
DBI
DBD::mysql

Required Files:
dmel.gff from flybase - (must be named in all lower case as shown, else rename in file)

Variables in script:
None needed to be changed for database population.
